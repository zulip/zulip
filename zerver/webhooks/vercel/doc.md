# Zulip Vercel integration

Get Zulip notifications for your Vercel deployments!

{start_tabs}

1. {!create-an-incoming-webhook.md!}

1. {!generate-webhook-url-basic.md!}

1. In Vercel, go to your team's **Settings**, and select the **Webhooks**
   tab. Click **Create Webhook**.

1. Set **Endpoint** to the URL generated above. Select the deployment events
   you want to be notified about, optionally choose which projects to watch,
   and click **Create Webhook**.

{end_tabs}
