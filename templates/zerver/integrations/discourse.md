# Zulip Discourse integration

Get Discourse notifications in Zulip!

{start_tabs}

1.  {!create-an-incoming-webhook.md!}

    !!! warn ""

        **Note:** The API keys for "Incoming webhook" bots are limited to
        only sending messages via webhooks. Thus, this bot type lessens
        the security risks associated with exposing the bot's API key to
        third-party services.

1.  Copy the `USERNAME` and `API KEY` - you'll need it later.

1.  Complete the
    [discourse-chat-integration setup instructions][setup-instructions].

{end_tabs}

{!congrats.md!}
