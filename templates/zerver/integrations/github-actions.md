# Zulip GitHub Actions integration

It's easy to send Zulip messages from GitHub Actions!

{start_tabs}

This is useful:

  1. If you want to report the success or failure of an Action to Zulip.

  1. If you want to send output from your actions to Zulip.

Instructions are available [here][instructions].
