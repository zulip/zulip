Keyboard shortcuts

Navigation
Down or j Next message
Up or k Previous message
/ Initiate a search
PgUp, PgDn Scroll up or down
Spacebar Scroll down
End Last message
Composing messages
c New stream message
C New private message
Enter or r Respond to message
R Respond to author
Tab then Enter Send message
Esc Cancel compose
Narrowing
s Narrow by stream
S Narrow by subject
v Narrow to all private messages
Esc Return to home view
