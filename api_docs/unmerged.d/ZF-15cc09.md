* [`POST /register`](/api/register-queue): Added `simplified_presence_events`
  [client capability](/api/register-queue#parameter-client_capabilities)
  to allow client to specify whether it supports `presence` events in the
  modern format.
* [`GET /events`](/api/get-events): For clients that support
  the `simplified_presence_events` [client capability](/api/register-queue#parameter-client_capabilities),
  the `presence` events will be send in the modern format by
  the server.
