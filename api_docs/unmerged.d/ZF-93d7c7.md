* [`GET /messages`](/api/get-messages): Added another type of anchor `date`
  along with the `anchor_date` parameter to fetch messages anchored to a
  specific date/time. If a date is provided, the datetime is set to
  midnight(00:00) on that day in UTC. If no timezone is provided, UTC is
  assumed. If no message has been sent on or after the `anchor_date`, we
  fall back to using `newest` anchor value.
