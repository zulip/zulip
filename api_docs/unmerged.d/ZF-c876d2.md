**Feature level ZF-c876d2**

* [`GET /calls/jitsi/create`](/api/create-jitsi-video-call): Added new
  endpoint that generates a Jitsi Meet video call URL with JWT authentication.

* [`POST /register`](/api/register-queue), [`GET /events`](/api/get-events):
  Added `server_jitsi_jwt_enabled` boolean field to the server settings
  data, indicating whether the server is configured for JWT authentication
  with a self-hosted Jitsi Meet server.
