* [`POST /register`](/api/register-queue): The
  [group permission setting](/api/register-queue#response) objects in
  `server_supported_permission_settings` now include a
  `replacement_group_name` field, naming the system group that the
  setting falls back to when the group configured for it can no longer
  be used (for example, when that group is deactivated, or when the
  only user in an anonymous group used for the setting is deactivated).
