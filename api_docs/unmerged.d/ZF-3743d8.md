* `PATCH /bots/{bot_user_id}`: Added `bot_type` parameter to support
  changing a bot's type.
* `PATCH /bots/{bot_user_id}`: Added the `service_name` parameter to
  support changing an incoming webhook bot's integration ID or bot's
  service name.
* [`GET /events`](/api/get-events): When a bot's type is changed via
  `PATCH /bots/{bot_user_id}`, the server now sends a `realm_user/update`
  event with the updated `bot_type` field. When the change involves a
  transition to or from a service bot type (outgoing webhook or embedded),
  a `realm_bot/update` event is also sent with the updated `services` field.
