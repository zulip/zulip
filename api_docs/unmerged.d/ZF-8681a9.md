  Added new `web_left_sidebar_view` user setting, controlling
  whether the web/desktop application's left sidebar shows the
  standard channel list or an inbox-style summary of conversations
  with unread messages. Possible values are `"channels"` and
  `"inbox"`; the default is `"channels"`.
