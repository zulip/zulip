## Import your organization from Discord into Zulip

### Export your Discord data

Discord does not provide a built-in message export feature, so the Zulip
import tool works with exports generated by
[DiscordChatExporter](https://github.com/Tyrrrz/DiscordChatExporter), a
popular third-party tool for exporting Discord servers.

<Steps>
  1. Install
     [DiscordChatExporter](https://github.com/Tyrrrz/DiscordChatExporter)
     by following its [installation
     instructions](https://github.com/Tyrrrz/DiscordChatExporter/blob/master/.docs/Getting-started.md).
  1. Obtain your Discord token by following the [DiscordChatExporter
     token
     guide](https://github.com/Tyrrrz/DiscordChatExporter/blob/master/.docs/Token-and-IDs.md).
  1. Export your server data in JSON format using DiscordChatExporter's
     **Export all channels** option. Make sure to:
     * Select **JSON** as the export format.
     * Enable **Download assets** to include avatars, attachments, and emoji.
     * Include direct messages if desired.
  1. You should end up with a directory containing JSON files for each
     channel and a `_media/` directory with all downloaded assets.
</Steps>

#### Import details

Whether you are using Zulip Cloud or self-hosting Zulip, here are a few notes to
keep in mind about the import process:

* Discord does not export workspace settings, so you will need to [configure
  the settings for your Zulip organization](/help/customize-organization-settings).
  This includes settings like [email
  visibility](/help/configure-email-visibility),
  [message editing permissions](/help/restrict-message-editing-and-deletion),
  and [how users can join your organization](/help/restrict-account-creation).
* Discord does not export user settings, so users in your organization may want to
  [customize their account settings](/help/getting-started-with-zulip).
* All users are imported with the Member [user role](/help/user-roles). Organization
  administrators can
  [change a user's role](/help/change-a-users-role) after the import.
* Discord text and voice channels are imported as Zulip channels.
* Discord threads are imported as topics within their parent channel.
  Messages not in a thread are given the topic "imported from Discord".
* Discord reply messages include a quote of the original message.
* System messages (e.g., user join notifications, pinned message
  notifications) are not imported.
* Message edit history is not imported.
* User avatars, attachments, custom emoji, and the server icon are
  imported from the local `_media/` directory in the
  DiscordChatExporter export.
