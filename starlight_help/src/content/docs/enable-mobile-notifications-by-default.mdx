Users can configure [mobile push notifications](/help/mobile-notifications)
on a [channel by channel](/help/channel-notifications) basis. For channels
where it's important that everyone stays on top of new messages, such as a
channel for urgent announcements, organization administrators can enable
mobile notifications by default for new subscribers.

Enabling mobile notifications by default does not change the notification
settings of anyone who is already subscribed to the channel, and users can
always [adjust their notification
settings](/help/channel-notifications#configure-notifications-for-a-single-channel)
for any channel.

## Enable mobile notifications by default for a channel

<FlattenedSteps>
  <NavigationSteps target="relative/channel/all" />

  1. Select a channel.

  <SelectChannelViewGeneral />

  1. Toggle **Mobile push notifications enabled by default**.

  <SaveChanges />
</FlattenedSteps>
