## Overview

[Zulip](https://zulip.com) is an open-source team collaboration platform that combines **topic-based threading** with real-time chat and email-style organization. It enables productive collaboration for both **live and asynchronous conversations**.  

Thousands of organizations, including Fortune 500 companies and leading open-source projects, rely on Zulip every day.  

**Key highlights:**  
- Topic-based threading for organized discussions  
- Works for both live and asynchronous collaboration  
- Active global development community with 1,500+ contributors  

Join our [development chat](https://zulip.com/development-community/) to connect with contributors and users worldwide.

## Getting Started

Zulip welcomes contributions of all kinds: code, documentation, translations, or feedback.  

### Contributing Code

- Start with the [guide for new contributors](https://zulip.readthedocs.io/en/latest/contributing/contributing.html).  
- Code is **highly readable, tested, and easy to modify**.  
- Over **185,000 words of contributor documentation** make onboarding easy.  

### Contributing Non-Code

- [Report an issue](https://zulip.readthedocs.io/en/latest/contributing/contributing.html#reporting-issues)  
- [Translate Zulip](https://zulip.readthedocs.io/en/latest/translating/translating.html)  
- [Give feedback](https://zulip.readthedocs.io/en/latest/contributing/suggesting-features.html)  

### Exploring Zulip

- Join the [Zulip community server](https://zulip.com/development-community/)  
- Learn about [Zulipâ€™s unique approach](https://zulip.com/why-zulip/) to organizing conversations  

### Running Your Own Zulip Server

- Self-host on **Ubuntu or Debian Linux**, via [Docker](https://github.com/zulip/docker-zulip), or using prebuilt images for [DigitalOcean](https://marketplace.digitalocean.com/apps/zulip) and [Render](https://render.com/docs/deploy-zulip)  
- Full self-hosting guide: [Zulip Self-Hosting](https://zulip.com/self-hosting/)  

### Using Zulip Without Hosting

- Explore [Zulip Cloud](https://zulip.com/plans/) hosting options  
- Zulip Cloud Standard is **free for eligible organizations**, including open-source projects  

### Participating in Outreach Programs

- Contribute through [Google Summer of Code](https://developers.google.com/open-source/gsoc) or [Outreachy](https://www.outreachy.org/)  

### Supporting Zulip

- Advocate for your organization to adopt Zulip  
- Become a [sponsor](https://github.com/sponsors/zulip)  
- Leave reviews in mobile app stores  
- Help others discover Zulip
